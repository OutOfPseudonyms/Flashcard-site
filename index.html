JavaScript and jQuery Definitions

JavaScript

the programming language of the web

Operators

a special character (or characters) that indicates an action to be performed

Variables

a container that stores some information such as a number, an object, or a string

Variable naming conventions

a set of rules for choosing the character sequence to be used for identifiers which denote variables, types, functions, and other entities in source code and documentationV

Functions

combine many instructions into a single line of code

Methods

A method is an action run on a piece of data; you can think of it as a message you send to a piece of data, and the result is the response

Arguments

provide a bit more information to methods to help them know what they're supposed to do

Parameters

a variable that's assigned to the argument

Return value

The return value is the method's response

Chaining methods

Calling a method directly on the return value of another method

Strings

a set of characters enclosed in quotations

Booleans

a statement returned as either true or false

Undefined

parameter with no assigned value yet.

NaN

not a number

Null

parameter with a value of noyhing

Alert

a function which pops up a dialog box with the string that you pass in as an argument

Comments

a convenient way to leave notes in your code for yourself or other programmers

jQuery

an extensive JavaScript library to help make our webpages interactive

Attributes

an identifying property of a data element, which may include but is not limited to data type, location, length, etc.
